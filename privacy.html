Kebijakan Privasi

Terakhir diperbarui: 26 November 2025

Selamat datang di KuloWallet. Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi serta data keuangan Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi informasi Anda saat menggunakan aplikasi mobile KuloWallet.

1. Informasi yang Kami Kumpulkan

Agar aplikasi dapat berfungsi dengan baik (seperti fitur Cloud Sync dan Laporan), kami mengumpulkan jenis informasi berikut:

  • Informasi Akun: Alamat email (wajib untuk login dan sinkronisasi data antar perangkat).
  • Data Keuangan: Data transaksi, kategori pengeluaran, dan anggaran yang Anda masukkan secara manual ke dalam aplikasi.
  • Data Perangkat: Informasi model perangkat dan versi sistem operasi untuk keperluan perbaikan bug (debugging) dan peningkatan performa.

2. Bagaimana Kami Menggunakan Informasi Anda

Kami menggunakan data tersebut semata-mata untuk:

  • Menyimpan dan mensinkronisasi catatan keuangan Anda ke Cloud Server kami yang aman.
  • Menghasilkan analisis keuangan (seperti fitur Survival Time Metric dan Early Warning System).
  • Membuat laporan (PDF) sesuai permintaan Anda.
  • Mengirimkan kode OTP atau notifikasi penting terkait akun (via layanan pihak ketiga seperti MailerSend).

Penting: Kami tidak menjual, menyewakan, atau membagikan data keuangan pribadi Anda kepada pihak ketiga untuk tujuan pemasaran (iklan).

3. Keamanan Data

Keamanan adalah prioritas utama kami. Kami menerapkan langkah-langkah teknis yang ketat:

  • Semua komunikasi data antara aplikasi dan server dienkripsi menggunakan protokol SSL/TLS.
  • Penyimpanan data sensitif di perangkat dilindungi menggunakan standar keamanan mobile terkini.
  • Akses ke database server dibatasi hanya untuk pemeliharaan sistem.

4. Layanan Pihak Ketiga

Aplikasi ini mungkin menggunakan layanan pihak ketiga untuk operasional teknis:

  • Cloudflare: Untuk keamanan jaringan dan DNS.
  • MailerSend/Email Provider: Untuk pengiriman email verifikasi/OTP.

5. Hak Pengguna & Penghapusan Data

Sesuai kebijakan Google Play Store, Anda memiliki hak penuh atas data Anda. Jika Anda ingin berhenti menggunakan KuloWallet dan ingin data Anda dihapus permanen dari server kami:

  • Anda dapat melakukan "Hapus Akun" langsung dari menu Pengaturan di dalam aplikasi.
  • Atau, Anda dapat menghubungi kami melalui email di bawah ini untuk permohonan penghapusan manual.

Setelah akun dihapus, seluruh data transaksi dan riwayat keuangan akan dimusnahkan secara permanen dan tidak dapat dikembalikan.

6. Hubungi Kami

Jika Anda memiliki pertanyaan mengenai privasi atau keamanan data di KuloWallet, silakan hubungi tim developer kami di:

Email: support@kulowallet.com

Privacy Policy

Last updated: November 26, 2025

Welcome to KuloWallet. We value your privacy and are committed to protecting your personal and financial data. This Privacy Policy explains how we collect, use, and protect your information when using the KuloWallet mobile application.

1. Information We Collect

To ensure the app functions properly (such as Cloud Sync and Reporting features), we collect the following types of information:

  • Account Information: Email address (required for login and data synchronization across devices).
  • Financial Data: Transaction data, expense categories, and budgets that you manually enter into the app.
  • Device Data: Device model information and operating system version for debugging and performance improvement purposes.

2. How We Use Your Information

We use this data solely to:

  • Store and synchronize your financial records to our secure Cloud Server.
  • Generate financial analysis (such as Survival Time Metric and Early Warning System features).
  • Create reports (PDF) upon your request.
  • Send OTP codes or important account-related notifications (via third-party services like MailerSend).

Important: We do not sell, rent, or share your personal financial data with third parties for marketing (advertising) purposes.

3. Data Security

Security is our top priority. We implement strict technical measures:

  • All data communication between the app and the server is encrypted using SSL/TLS protocols.
  • Sensitive data storage on the device is protected using the latest mobile security standards.
  • Access to the server database is restricted to system maintenance only.

4. Third-Party Services

This app may use third-party services for technical operations:

  • Cloudflare: For network security and DNS.
  • MailerSend/Email Provider: For sending verification emails/OTP.

5. User Rights & Data Deletion

In accordance with Google Play Store policies, you have full rights over your data. If you wish to stop using KuloWallet and want your data permanently deleted from our servers:

  • You can perform "Delete Account" directly from the Settings menu within the app.
  • Or, you can contact us via the email below for a manual deletion request.

Once the account is deleted, all transaction data and financial history will be permanently destroyed and cannot be recovered.

6. Contact Us

If you have questions regarding privacy or data security at KuloWallet, please contact our developer team at:

Email: support@kulowallet.com
